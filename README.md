# ZTS Platform — status site

**Live site: https://szagar.github.io/zts-status/**

An automated options-trading platform: market data, strategy bots, order management, and a canonical trade ledger — built as a Python monorepo.

This repository holds the published static snapshot of the site — roadmap
position, daily system-health checks, paper-trading activity, and screenshots.

Everything here is **machine-generated**: a scheduled job rebuilds the whole
tree, runs it through a redaction gate, and force-pushes a single fresh commit
each weekday afternoon (US/Eastern). History is intentionally one commit deep,
and manual edits or pull requests will be overwritten by the next publish.
